# Microbiome Explorer

A simple Next.js + TypeScript app that shows a list of skin microorganisms with filtering by type and sorting by abundance.

## Features

- Display microorganisms with basic information
- Filter by type (bacteria / fungi)
- Sort by relative abundance
- Simple and clean UI

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript

## Run locally

```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000).
